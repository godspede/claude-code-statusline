# Contributing

Thanks for your interest! This is a small, single-purpose project.

- **Bugs / ideas:** open a GitHub issue. For rendering bugs, please paste the
  offending statusline line and your `pwsh --version` (or `$PSVersionTable`).
- **Pull requests** are welcome. Keep changes focused and match the existing
  style.
- **Two ground rules:** no external dependencies (no jq, no modules, no
  network calls beyond the existing `oauth/usage` lookup), and it must stay
  Windows / PowerShell friendly (PowerShell 7+ and Windows PowerShell 5.1).
